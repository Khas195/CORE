Detailed Description

This class handle all movement related behaviors in 3D The host object (mentioned in blow) is the object which this script will perform its functions on .

!– The host object is not necessary be the parent gameobject of the script.!–

◆ Step()

void Movement.Step ( float  speed,
int  forward,
int  side,
bool  isAirborne 
)
private

This function move the host object according to the forward and side inputs of the player.

The movement is performed by manipulating the velocity property of the rigidbody. The speed of the movement is reduced if the host object is airborned.

Parameters
speedis the desired movement speed for the host object.
forwardis how much the player need to move forward or backward.
sideis how much the player need to move sideway.
isAirborneis whether the host object is in the air or not.
